My Flask Webseit
It has been make with **Python**, **Flask**, **SQLalchemy**, **Bootstrap**.It features a secure user authentication system and allows users to browse and select tickets across different towns in Austria.

## Tech Stack
* **Language**: Python 3.12.3
* **Framework**: Flask
* **Database**: SQLalchemy with SQLite
* **Frontend**: HTML5 and Bootstrap

## What can it do?

* **User Authentication**: Secure user Login and Logout functionality.
* **Ticket Booking System**: Users can browse and select tickets in various Austrian towns.
* **User Dashboard**: Users can see exactly which tickets they have chosen and the location for each ticket.
* **Relational Database**: Dynamic data rendering and storage managed via SQLAlchemy models.
